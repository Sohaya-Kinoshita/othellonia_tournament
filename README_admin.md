# 管理画面（初心者向け）使い方

このドキュメントは、非技術者向けに「対戦結果の登録・管理」を簡単に行う手順を説明します。

## 目的

- ブラウザから対戦結果を追加・編集・削除し、サイトの`results.html`に表示できるようにすること。

## 管理画面にアクセス

- ブラウザで `admin.html` を開きます（例: `file:///.../src/admin.html` またはホスト先のURL）。

## 簡単ログイン（初心者向け）

- 管理画面に「テストログイン（管理者）」というボタンがあります。これは**テスト用**です。
- このボタンを押すと一時的に管理者としてログインし、編集機能が使えるようになります。
- 本番運用ではこのボタンを使わず、正式なログインを行ってください。

## 対戦結果の追加手順

1. 「テストログイン（管理者）」を押す（ログイン済みであれば不要）。
2. 「時間」「対戦1」「対戦2」「得点」「メモ」を入力します。
3. 「追加 / 保存」ボタンを押すと、一覧に行が追加されます。

## 編集・削除

- 一覧に表示される各行の「編集」ボタンでフォームへ読み込み、編集後に再度「追加 / 保存」します（編集は実際には既存行を一度削除してから再追加します）。
- 「削除」ボタンで該当行を削除できます。

## データの永続化と移行（初心者向け）

- 編集内容はブラウザの`localStorage`に保存されます。別のPCやブラウザに移す場合は「JSONをダウンロード」でファイルを保存し、そのファイルを別の環境で「インポート」してください。
- 管理者が複数いる場合や中央管理したい場合は、サーバー保存（私が実装します）に移行することをおすすめします。

## よくある質問

- Q: 編集した内容が消えた
  - A: 同じブラウザ・同じPCのままなら`localStorage`が保持されます。別のPCやブラウザでは表示されません。移行するにはJSONをエクスポートしてインポートしてください。

- Q: テストログインは安全ですか？
  - A: テストログインは利便性のための簡易機能です。本番での使用は推奨しません（誰でも押せば管理者になります）。

## 次のステップ（必要なら）

- サーバー側に保存して複数管理者で同期したい場合、簡易のPOST APIとファイル保存、またはクラウドDB（Firestore/SQLite等）への移行を実装できます。ご希望があれば提案します。
