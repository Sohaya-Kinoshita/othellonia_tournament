# 出場チーム管理ファイル

## ファイル説明

`teams.xlsx` は出場チームの情報を管理するExcelファイルです。

## セットアップ方法

### 1. openpyxlのインストール

ターミナルで以下のコマンドを実行してください：

```bash
pip install openpyxl
```

### 2. Excelファイルの生成

ターミナルで以下のコマンドを実行：

```bash
python create_teams_excel.py
```

すると `data/teams.xlsx` が生成されます。

## 使い方

### Excelで編集

1. `teams.xlsx` をダブルクリックして開く
2. 必要な情報を入力
3. 保存

### 編集項目

| 項目         | 説明                                   | 例              |
| ------------ | -------------------------------------- | --------------- |
| チーム名     | 出場するチームの名前                   | サンプルチーム1 |
| チーム代表者 | 代表者の氏名                           | 田中太郎        |
| メンバー数   | 参加予定のメンバー数                   | 4               |
| 参加予定     | 〇=参加予定、×=不参加                  | 〇              |
| 連絡先       | 代表者の連絡先（電話番号またはメール） | 090-1234-5678   |
| メモ         | その他の備考                           | 新チーム        |

## ファイルの場所

`/data/teams.xlsx`

## 注意

- Excelファイルはヘッダー行（1行目）は編集しないでください
- サンプルデータは削除して実際のチーム情報を入力してください
